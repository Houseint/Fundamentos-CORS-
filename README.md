# HELP.TXT
Este arquivo fornece instruções sobre como configurar e executar o projeto "Fundamentos CORS". Siga os passos abaixo para iniciar o servidor backend e o frontend.

Na pasta backend instale os packages: npm init -y & npm i express cors
Na pasta frontend instale os packages: npm init -y & npm i serve

Navegue até a pasta do `backend`, inicie o server com o comando:
```
node server.js
```
Para iniciar o frontend, navegue até a pasta `frontend` e execute:
```
npx serve -l 3000
```
